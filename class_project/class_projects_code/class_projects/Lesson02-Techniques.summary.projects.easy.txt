# Class Projects
- **Title:** Image Classification with CIFAR-10
- **Difficulty:** Easy
- **Tech description:** Build a supervised learning model to classify images from the CIFAR-10 dataset using convolutional neural networks (CNN).
- **Project idea:** Students will implement a simple CNN to classify images into 10 different categories (e.g., airplane, car, bird, etc.). They will preprocess the images, create the model architecture, train the model, and evaluate its performance using accuracy and confusion matrix metrics.
- **Data set to use:** CIFAR-10 (available at https://www.cs.toronto.edu/~kriz/cifar.html)
- **Python libs:** TensorFlow/Keras, NumPy, Matplotlib, Scikit-learn
- **Links to relevant tools:**
  - Keras Documentation: https://keras.io/
  - TensorFlow Guide: https://www.tensorflow.org/guide
- **Links to related resources:**
  - CIFAR-10 Dataset Explained: https://www.analyticsvidhya.com/blog/2021/06/understanding-the-cifar-10-dataset/
  - Convolutional Neural Networks for Beginners: https://towardsdatascience.com/convolutional-neural-networks-for-beginners-39e46f8e1dc4
