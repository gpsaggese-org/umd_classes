# Related Python Packages
- Package: Scikit-learn
  - Description: A widely-used library that provides simple and efficient tools for data mining and machine learning in Python. It offers support for both supervised and unsupervised learning, as well as various utilities for preprocessing, model evaluation, and hyperparameter tuning.
  - Website: [scikit-learn.org](https://scikit-learn.org/)
  - Documentation: [Scikit-learn Documentation](https://scikit-learn.org/stable/documentation.html)

- Package: TensorFlow
  - Description: An open-source platform for machine learning that offers flexible tools, libraries, and community resources to build and deploy machine learning applications. Particularly strong in deep learning and neural networks, it supports both supervised and unsupervised learning paradigms.
  - Website: [tensorflow.org](https://www.tensorflow.org/)
  - Documentation: [TensorFlow Documentation](https://www.tensorflow.org/learn)

- Package: Keras
  - Description: A user-friendly neural network library that runs on top of TensorFlow. It allows for easy and fast prototyping, scaling, and deploying of deep learning models using a simple API for various learning paradigms including supervised and reinforcement learning.
  - Website: [keras.io](https://keras.io/)
  - Documentation: [Keras Documentation](https://keras.io/guides/)

- Package: PyTorch
  - Description: An open-source machine learning library that emphasizes flexibility and speed, making it ideal for deep learning developments and research. It supports dynamic computation graphs and a rich ecosystem for building models in reinforcement learning and computer vision.
  - Website: [pytorch.org](https://pytorch.org/)
  - Documentation: [PyTorch Documentation](https://pytorch.org/docs/stable/index.html)

- Package: LightGBM
  - Description: A gradient boosting framework that uses tree-based learning algorithms. It's designed for distributed and efficient training, providing state-of-the-art performance for various machine learning tasks, including classification and regression.
  - Website: [lightgbm.readthedocs.io](https://lightgbm.readthedocs.io/en/latest/)
  - Documentation: [LightGBM Documentation](https://lightgbm.readthedocs.io/en/latest/Documentation.html)
